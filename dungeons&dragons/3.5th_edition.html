3.5th Edition

Back in the days of 3.0, the game released its d20 system to public use which spawned other great tabletops, including the wildly popular Pathfinder. 3.5 revised this edition to make for some better balancing and put forth some tweaks, and 5 came out with the intention to make the game more accessible to new players.

In total, there are 11 different character sheets for D&D, and that's one for each of the core character classes at that time. There's also a d20 Modern character record sheet, a game that used the 3.5 D&D rules and applied them to a modern setting.
